# Tradfri-Cast-Control
Use a five button Tradfri remote to control your cast device.

Buttons are mapped to the following controls:

**Center button:**
* Play/pause
* Power toggle (HOLD)

**Left button:**
* Seek negative
* Previous track (HOLD)

**Right button:**
* Seek positive
* Next track (HOLD)

**Up button:**
* Volume up

**Down button:**
* Volume down
* Mute toggle (HOLD)
